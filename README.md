## AWSをはじめようは「DNSをはじめよう」の続き

「DNSをはじめよう」で自分のドメインを買って、
Route53をネームサーバとして使うところまでやったので、
今度は「AWSをはじめよう」でサーバを立てて
WordPressで自分のサイトを作って公開するところまでやってみましょう。
